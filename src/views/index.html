Timestamp Microservice

Convert any valid date or Unix timestamp to both Unix and UTC formats

How It Works

This microservice accepts a date or a Unix timestamp as a parameter in the URL and returns the corresponding Unix time and UTC format as a JSON response.

For example, you can try entering a date string or Unix timestamp directly in the API URLs below to get the conversion results:

Example Usage:

  • [project url]/api/2015-12-25
  • [project url]/api/1451001600000

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}

Note:

The date can be provided in YYYY-MM-DD format, and the timestamp should be in milliseconds (Unix time).

This project serves as a basic API microservice to demonstrate date-time conversion.

Important: This is purely for demonstration and testing purposes on the API level.
